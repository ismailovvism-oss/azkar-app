# Азкары — публичная версия (`azkar-app`)

Простое приложение-помощник для утренних, вечерних и ситуативных
азкаров: открыл, прочитал, отметил касанием. Прогресс и личные
заметки хранятся прямо в браузере, без сервера и без регистрации.

Это **публичная** сборка. Её код (`azkary.html`) идентичен семейной
версии — отличие только в папке `personal/`:

- здесь один файл `personal/default.json` с `"mode": "public"`;
- приложение открывается по обычной ссылке **без** `?u=`;
- при первом запуске спрашивает имя и запоминает его локально;
- вверху главного экрана — предисловие автора (поле `letter`);
- есть экран **«Как пользоваться»** (справка) и кнопка
  **«Предложи свой зикр или функцию»** (пишет на адрес из
  `feedbackEmail`).

## Структура

```
azkary.html            ← всё приложение (vanilla JS, один файл)
content.json           ← зикры (разделяется с семейной версией)
personal/default.json  ← публичная учётка (mode: public)
manifest.json, icon.svg, _redirects
```

## Локальный запуск

```
python3 -m http.server 8080
# открыть http://localhost:8080/azkary.html
```

Двойной клик по файлу не сработает — браузер блокирует чтение
локальных файлов по схеме `file://`.

## Деплой

Перетащи папку в [Netlify Drop](https://app.netlify.com/drop) как
отдельный сайт. `_redirects` отдаёт `azkary.html` с корня.

## Синхронизация с семейной версией

`azkary.html` и `content.json` — общие. После правок в основном
репозитории `azkary` синхронизируй их сюда (в основном репозитории
есть скрипт `sync-public.sh`). Папка `personal/` между версиями
**не** синхронизируется.
